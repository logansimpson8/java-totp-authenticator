## Java TOTP Authenticator
***
A minimal dependency java library that allows you to implement TOTP (Time Based One Time Passwords) based on RFC 6238. 

compatible with google authenticator and Authy (confirmed tested)

## Features
***
• Custom Engine - I have created my own math engine to handle the bit shifting and dynamic truncation

• Time Sync - Uses Unix Epoch time to generate 30 second windows without sync issues across devices

• Base 32 - This Authenticator generates a secret string compatible with industry standard base 32 

• Minimal Dependencies - built using only apache commons for base 32 decoding, all core cryptographic logic handled manually 

• QR code - the system automaitcally generates a link to a qr code that can be scanned using google authenticator to quickly add the secret string much more efficiently with a pre filled name of choice.

## How does it work?
***
• Hashing - it works by mashing a secret key generated by the system and the current time which constantly is changing through a HMAC-SHA1 blender. 

• Dynamic Truncation - the hash will consistnetly be changing, and so the system uses the last 4 bits within the current hash as an offset, this then allows the systsem to pick a new starting point each time increasing security. 

• Bit Shifting - it then extracts 4 bytes and connects them into a 32 bit integer using <<. 

• Modulo Math - it then preforms %1,000,000 to turn it into a readable 6 digit pin

## Usage/testing
***
Run demo.java to generate a secret code. you can then add this code to your authenticator and test the system.
